Introdução

Bem-vindo ao painel administrativo da Visage Distribuidora. Este sistema foi desenvolvido para simplificar a gestão completa do seu negócio, desde o controle de produtos e estoque até o acompanhamento de pedidos e clientes.

Use a barra lateral esquerda para navegar entre as diferentes seções do sistema. O dashboard inicial apresenta uma visão geral do seu negócio com estatísticas importantes e alertas que precisam da sua atenção.

Gerenciamento de Produtos

Cadastro de Produtos

  1. Acesse Estoque > Produtos no menu lateral
  2. Clique no botão Criar no canto superior direito
  3. Preencha os campos obrigatórios:
    • Nome: Nome do produto que será exibido para os clientes
    • Preço: Valor de venda do produto
    • Custo: Valor de aquisição (para cálculo de lucro)
    • SKU: Código único do produto (importante para controle)
  4. Os campos opcionais podem ser preenchidos para maior detalhamento:
    • Descrição: Informações detalhadas sobre o produto
    • Imagem: URL da imagem para exibição
    • Categoria: Agrupamento para organização
  5. Para configurar o controle de estoque:
    • Quantidade em Estoque: Estoque inicial
    • Nível Mínimo: Quando o sistema deve alertar sobre baixo estoque
    • Nível Máximo: Capacidade máxima recomendada
  6. Clique em Salvar para cadastrar o produto
Dica: Use SKUs padronizados para facilitar a gestão. Por exemplo, para shampoo da marca X, use X-SH-001.

Edição e Exclusão

Para editar um produto existente:

  1. Localize o produto na lista de produtos
  2. Clique no ícone de edição (lápis) na coluna de ações
  3. Atualize os campos necessários e clique em Salvar

Para excluir um produto:

  1. Localize o produto na lista de produtos
  2. Clique no ícone de exclusão (lixeira) na coluna de ações
  3. Confirme a exclusão na janela de diálogo
Atenção: A exclusão de produtos só deve ser feita se o produto não tiver histórico de vendas ou movimentações. Caso contrário, é recomendado apenas desativar o produto marcando "Em estoque" como "Não".

Controle de Estoque

Ajuste de Estoque

Para ajustar o estoque de um produto:

  1. Acesse Estoque > Gerenciar Estoque no menu lateral
  2. Selecione o produto que deseja ajustar no menu suspenso
  3. Informe a quantidade a ser ajustada:
    • Valores positivos: Para adicionar ao estoque (ex: +10)
    • Valores negativos: Para remover do estoque (ex: -5)
  4. Informe o motivo do ajuste para registro no histórico
  5. Clique em Salvar Ajuste
Dica: Sempre registre um motivo detalhado para facilitar o rastreamento das alterações e auditorias futuras.

Movimentações de Estoque

Para visualizar o histórico de movimentações:

  1. Acesse Estoque > Movimentações no menu lateral
  2. Use os filtros para localizar movimentações específicas:
    • Por produto: Digite o nome do produto no campo de pesquisa
    • Por tipo: Filtre por entrada (increase) ou saída (decrease)
    • Por data: Use os filtros de data para períodos específicos

Cada registro mostra o produto, quantidade alterada, estoque anterior, estoque atual, data e motivo da movimentação.

Alertas de Estoque

O sistema gera alertas automaticamente quando:

  • O estoque de um produto fica igual ou abaixo do nível mínimo configurado
  • Um produto fica sem estoque (zerado)

Estes alertas são exibidos no Dashboard principal e podem ser gerenciados nas seguintes formas:

  1. Adicionar mais estoque através da página Gerenciar Estoque
  2. Ajustar o nível mínimo do produto se o valor atual não for adequado

Gestão de Pedidos

Visualização de Pedidos

Para visualizar todos os pedidos:

  1. Acesse Vendas > Pedidos no menu lateral
  2. Os pedidos são listados em ordem cronológica, com os mais recentes no topo
  3. Utilize os filtros para encontrar pedidos específicos:
    • Por cliente: Digite o nome do cliente
    • Por status: Filtre por status (pendente, concluído, etc.)
    • Por data: Selecione um período específico
  4. Clique em um pedido para ver seus detalhes completos

Atualização de Status

Para atualizar o status de um pedido:

  1. Clique no pedido que deseja atualizar
  2. Na página de detalhes, altere o campo Status para uma das opções:
    • Pendente: Pedido recebido, aguardando processamento
    • Em processamento: Pedido está sendo preparado
    • Enviado: Pedido foi despachado para o cliente
    • Concluído: Pedido entregue e finalizado
    • Cancelado: Pedido foi cancelado
  3. Clique em Salvar para atualizar o status
Importante: Ao cancelar um pedido, o sistema não restaura automaticamente o estoque. Se necessário, faça um ajuste manual para retornar os itens ao estoque.

Relatórios e Análises

Dashboard

O dashboard principal apresenta uma visão geral do seu negócio, incluindo:

  • Total de produtos e quantos estão em estoque
  • Valor do estoque em custo e preço de varejo
  • Alertas de estoque para produtos abaixo do nível mínimo
  • Total de pedidos e quantos estão pendentes
  • Movimentações recentes de estoque

Acesse o dashboard a qualquer momento clicando em Dashboard no menu lateral.

Exportação de Dados

Para exportar dados do sistema:

  1. Navegue até a lista que deseja exportar (Produtos, Pedidos, Movimentações)
  2. Aplique os filtros desejados para refinar os dados
  3. Clique no botão Export no canto superior direito
  4. Selecione o formato de exportação desejado (CSV, Excel, PDF, etc.)
  5. O arquivo será gerado e baixado automaticamente
Dica: Exporte regularmente seus dados para backup e análises em ferramentas externas como Excel ou Google Sheets.

Dicas e Melhores Práticas

Gestão de Estoque
  • Faça contagens físicas periódicas para reconciliar o estoque
  • Configure níveis mínimos adequados para cada produto
  • Registre sempre o motivo das movimentações manuais
  • Verifique regularmente os alertas de estoque baixo
Organização de Produtos
  • Use SKUs consistentes para fácil identificação
  • Categorize os produtos adequadamente
  • Mantenha as descrições dos produtos completas
  • Use imagens de qualidade para melhor visualização
Processamento de Pedidos
  • Atualize o status dos pedidos em tempo real
  • Verifique diariamente os pedidos pendentes
  • Entre em contato com o cliente para informações incompletas
  • Documente qualquer alteração no pedido original
Segurança e Backup
  • Exporte regularmente seus dados para backup
  • Não compartilhe suas credenciais de acesso
  • Faça logout ao terminar de usar o sistema
  • Verifique regularmente o log de atividades
